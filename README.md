# Cat Facts

## Description
An educational single-page application that retrieves and presents random facts about cats at the click of a button.

## APIs Used
- [CatFact Ninja](https://catfact.ninja/)

## Features Implemented
- Asynchronous data fetching to update the UI without reloading.
- Modern, accessible UI with high-contrast text and a vibrant lime-to-yellow gradient.
- Responsive card-based design for mobile and desktop viewing.
